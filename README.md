AirMCU 是一个由社区推动，旨在适配上海合宙通信有限公司的 Arm-Cortex 系列 MCU 的 Arduino 框架。

## 文档与上手教程

可以直接查看 [AirMCU 的官方文档](https://arduino.luatos.com/)，包括了接口资料与上手教程。

## 支持芯片

目前我们支持的 MCU 如下：

| MCU 系列  | 是否支持 |      详细链接       |
| :-------: | :------: | :-----------------: |
|  Air001   |    ✅     | <https://air001.cn> |
| Air32F103 |    🔨     | <https://air32.cn>  |

有关开发板的更多资料，请查看所述链接中的开发板内容。
